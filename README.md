# CZEN - Commit ZEN
Expired Study Project on [Committizen](https://github.com/commitizen-tools/commitizen)
## Requirements
- [Git] `^2.`+
## Usage
### Help
Run in your terminal

```bash
czen help
```

or the shortcut

```bash
czen h
```
### Committing

```bash
czen commit
```

or the shortcut

```bash
czen c
```

### Rollback to Tag

```bash
czen rollback
```

or the shortcut

```bash
czen r
```
### Show tags

```bash
czen tag
```

or the shortcut

```bash
czen t
```
Select the tag for rollback and confirm
